Emuns I Have Learned To Use Cheat Sheet
Method Description Example Returns
count Returns the numbers of times the emun is counted in an array [1,2,4,2].count(2) 2, since 2 is shown twice in the array
drop Drops first n element from the emun and returns in the rest of the elements in an array [1,2,3,4,5,0].drop(3) [4,5,0]
find Passes each entry in emun to block. Returns the first for which is the block is true (1..100).find {|i| i % 5 == 0 and i % 7 == 0} 35
find_all Returns an array containing all elements of emun for which the given block returns true (1..10.find_all { |i| i % 3 == 0 } [3,6,9]
include? Returns true if any member of emun equals object. Equality is tested using ==. ["a","b","d"].include?("d") true
map Returns a new array with the results of running the block once for every element in emun [1,2,3].map {|i| i*2 } [2,4,6]
