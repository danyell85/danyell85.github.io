Group_By Methods

The group_by method allows you to group items in your array by a condition and return in a hash.
Example:
pet_names=["Joe","Cassie","Bob","Dylaan"] pet_names.group_by{|name| name.length}
The name(item) to run through the method will be grouped by the length of the pets name. All names with the length of 3 will be returned as the value in the hash with the key being 3. Cassie and Dylaan will be grouped together because their length is 6.
