Getting Classy With Ruby

Forgive me as you read this. I am still rough around the edges with classes or more specifically when to use instance variables. This blog post is a work in progress.

What is a Class
A class is a roadmap from which individual objects are created. And let's not forget the a class is also an object in Ruby, basically everything is. Within a class you can have methods for the certian characteristics you want you class to have. Example: You have 3 cats but they all have a different combination of color, names and genders. Your class would be cat, the individual methods to help set them apart would be color, names and gender.
Instance Variables/
An instance variable definded by placing and @ sign infront of the name is making it avaible to every method in the class. It is allowing it to move freely within the class and hold the content at the moment/instance. Local methods are defined within a method and cant transfer to another without redefining it.
