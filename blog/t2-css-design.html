Blocks In CSS

Blocking in CSS allow for items to be placed side by side. Depending on how you want your webpage to look there are a couple of options when using blocking.
1. Inline blocking places the items in a line and keeps the height and width that they were programmed with. display:inline-blocking;
2. Inline will place the itmes in the same row but override the dimensions and will make them all the same size. display:inline;
