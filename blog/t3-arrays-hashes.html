Arrays & Hashes

Arrays
Arrays are ordered collections of any object. Objects can be strings, ints, floats or other arrays. The arrays orders itself in an integer-index. Integers start at 0 so to find the first object in an array the user can input array[0].
Hashes
Hashes are similar to arrays. They are a collection of ojects and there values. Unlike arrays, hashes allow use an object type index. It is like a dictionary. When looking up a work(key in hash) there is a definition next to the word(value for hash).To search for something in a hash the user can input hash["water"].
