Static Vs Dynamic

Static Languages

Programming languages that have static typing have to have the variables declared for which type they are. For example if you wanted to declare a variable as an integer value you first would need to tell the variable it was going to be an integer then give it a value:

int number1; number1 = 5;

Dynamic Languages

Other languages have dynamic typing which allows you to for go the declartion of what type of variable it will be. Example:
