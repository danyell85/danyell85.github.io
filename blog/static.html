Static Vs Dynamic

Static Languages

Programming languages that have static typing have to have the variables declared for which type they are.These languages the variables are known at the time of compliling. This allows for the compiler to check and catch bugs early on. For example if you wanted to declare a variable as an integer value you first would need to tell the variable it was going to be an integer then give it a value: int number1; number1 = 5;

Dynamic Languages

Other languages have dynamic typing which allows you to forgo the declartion of what type of variable it will be. The variable type is interperted during runtime. This allows for the programmer to take out some of the declartion code and be faster. Example: number 1 = 5;
