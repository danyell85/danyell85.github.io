Partition

I ran accross this handing method during a pairing session this week. Partition returns two arrays after iterating through a range or even another array. The first array will contain the elements for what parition evaluated as true and the rest get placed within the second array. Let me show you an example to further explain.

Example 1:Calling parition on a range of numbers seperate the odd elements from the even. (1..10).partition {|element| element.odd?} => [[1,3,5,7,9],[2,4,6,8,10]] Example 2: Calling parition on an array of numbers to seperate the elements that are evenly divisble by 3. array_of_numbers = [1,3,5,7,9,12] array_of_numbers.partition{|element| if element % 3 == 0}
